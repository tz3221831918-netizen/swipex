# SwipeX – Window Snap for macOS

SwipeX is a trackpad-first macOS window manager for people who want window control to feel close to the window itself.

Official website: https://swipex.lynvivian.com/

SwipeX lets you manage Mac windows from the title bar. Instead of memorizing a large shortcut map or dragging windows to screen edges, you can use title-bar swipe gestures to snap, maximize, reveal, and restore windows in a way that feels spatial and direct.

## What SwipeX Does

- Title-bar swipe snapping for halves, corners, thirds, and two-thirds.
- Smart Fill with Option-Swipe to use the available space around existing windows.
- Counterclockwise circle gesture to maximize a window without entering a separate macOS full-screen Space.
- Clockwise circle gesture to show the useful window behind the current one.
- Window Layouts to save and restore repeated work setups such as Writing, Coding, Research, and Support.
- Quick Access for apps, websites, folders, and files without memorizing a large shortcut map.
- Multiple-display workflows for Mac users who move between laptop screens and external displays.

## Why It Exists

Window management is a small task, but it repeats all day. A few seconds spent dragging, resizing, finding hidden windows, or rebuilding the same workspace can quietly interrupt focus.

SwipeX is built around a simple idea: window management is spatial, so the interaction should feel spatial too.

Put the pointer on the title bar. Swipe in the direction the window should go. Save layouts when the whole workspace matters. Keep common apps, folders, files, and websites close through Quick Access.
