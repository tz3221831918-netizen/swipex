# SwipeX Support

SwipeX is a macOS window manager built around title-bar swipe gestures, Window Layouts, Smart Fill, and Quick Access.

## Official Links

- Website: https://swipex.lynvivian.com/
- Support: https://swipex.lynvivian.com/support
- Privacy: https://swipex.lynvivian.com/privacy
- Demo: https://youtu.be/qHaV9uiBXlM

## Common Notes

SwipeX uses standard macOS permissions so it can observe gestures and move windows. If window snapping does not work as expected, check that the app has the requested macOS permissions and that the target window is not restricted by its own minimum size.
